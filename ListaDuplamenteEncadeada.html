Lista Duplamente Encadeada

O que é uma Lista Duplamente Encadeada?

Uma lista duplamente encadeada é uma estrutura de dados que consiste em uma sequência de nós, onde cada nó armazena um item, uma referência ao próximo nó e uma referência ao nó anterior. Diferente de uma lista simplesmente encadeada, em uma lista duplamente encadeada, cada nó possui dois ponteiros, permitindo a navegação tanto para frente quanto para trás na lista.

Como a Lista Duplamente Encadeada Funciona?

Em uma lista duplamente encadeada, cada nó contém três elementos principais:

  • O item a ser armazenado
  • Um ponteiro para o próximo nó
  • Um ponteiro para o nó anterior

Essa estrutura permite a navegação bidirecional na lista. Você pode percorrer a lista a partir do início (head) ou do final (tail), seguindo os ponteiros para frente e para trás.

Operações Básicas em uma Lista Duplamente Encadeada

As operações básicas em uma lista duplamente encadeada são semelhantes às da lista simplesmente encadeada e incluem:

  • Inserção no início da lista
  • Inserção no final da lista
  • Inserção após um elemento específico
  • Remoção de um elemento
  • Pesquisa por um elemento
  • Travessia da lista

Vantagens e Desvantagens da Lista Duplamente Encadeada

Vantagens

  • Navegação bidirecional eficiente
  • Melhor desempenho em operações que exigem acesso aos elementos vizinhos

Desvantagens

  • Ocupa mais espaço em memória devido aos ponteiros extras
  • Complexidade maior de implementação em comparação com a lista simplesmente encadeada
